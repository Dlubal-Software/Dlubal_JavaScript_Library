# setMainHingeValues(member_hinge, values, property_1, property_2) ⇒

Sets values to member hinge (private)

Kind: global function
Returns: Returns modified member hinge

Param Type Description
member_hinge Object Member hinge
values Array Values to be set, [Translational/Rotational, Spring constant, Nonlinearity] - Values can be in two formats: [bool, float, int] - if bool is true (translation is enabled), then can be specified next two values (spring constant and nonlinearity) [bool, int] - if bool is false (translation is disabled), then can be specified only next one value (nonlinearity) Nonlinearity: can be string name or index: None (0), Fixed if negative (1), Fixed if positive (2), Failure all if negative (3), Failure all if positive (4), Partial activity (5), Diagram (6), Stiffness diagram (7), Friction direction 1 (8), Friction direction 2 (9), Friction direction 1 2 (10), Friction direction 1 + 2 (11)
property_1 String Spring constant string name
property_2 String Nonlinearity string name
